# Austin Sensors Map

This repository hosts an interactive map of PurpleAir sensors and AQS stations in the Austin five-county area.

View the map here:

https://hhallaji.github.io/Austin_Sensors_map/

## What the Map Shows

- PurpleAir sensor locations
- Sensor status, including online, offline, and other installation/status categories
- AQS station locations
- 1 km radius circles around each AQS station
- Distance from each sensor to its nearest AQS station
- Highlighting of the nearest AQS station when a sensor is clicked
- Filters for exploring sensors by status and data coverage

## Notes

The map was created from the available sensor list and AQS station information. Some sensors may be excluded if they do not have usable coordinates or enough address information to estimate coordinates.

The map is intended as an exploratory tool to help identify sensor locations, status, proximity to AQS stations, and possible priorities for installation, reinstallation, or historical data checks.
